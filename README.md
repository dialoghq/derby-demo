derby-passport
==============

Running
-------

This app can be launched using node-foreman.

install it with
    npm i -g foreman

run it with
    $ nf start

Developing
----------

 1. grunt will watch for changes, compile coffee into js, and launch server
    install it with
        npm i -g grunt
    run it with
        $ grunt
